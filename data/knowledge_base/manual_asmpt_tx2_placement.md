# ASMPT SIPLACE TX2 Placement Head Technical Manual

**Document ID:** MAN-SMT-TX2-2026  
**OEM Vendor:** ASMPT Placement Solutions  

## 1. Vision System & Head Calibration
- **Placement Rate:** Up to 96,000 components per hour (CPH).
- **Fiducial Camera Resolution:** 5 Megapixel telecentric camera with 8-segment RGB illumination ring.
- **Placement Force:** Adjustable 0.5 N to 5.0 N ± 0.1 N.

## 2. Vacuum Nozzle Overhaul Procedure
- **Vacuum Filter Change:** Clean 10-micron sintered bronze filter every 200 hours.
- **Nozzle Pitch Offset:** Run auto-calibration sequence using glass target GT-50.
